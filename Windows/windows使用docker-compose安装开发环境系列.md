## 环境准备

Docker安装教程：在Docker目录下
> 注：建议使用`Git Bash Here`执行以下命令
```shell script
# 创建文件夹
mkdir -p D:/Linux/Docker
cd D:/Linux/Docker

git clone https://github.com/ZhaoTonApple/Devops.git
cd Devops/Windows
```

## 运行服务

> 环境部署见每个服务下的`run.md`
> ex: `Windows/portainer/run.md`
